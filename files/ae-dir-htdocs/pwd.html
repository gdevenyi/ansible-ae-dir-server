Intended Audience:
All persons with an Æ-DIR personal user account

This document describes how to use the Æ-DIR password self-service which implements the following use-cases:

  1. Change own password
  2. Check own password
  3. Password reset process

Change own password

This function allows a user to change his/her password of an Æ-DIR personal user account.

Hints:

  • You must not use old passwords (checked against password history).
  • Your password MUST be sufficiently long (read your password policy).
  • You cannot change your password again for a certain period of time after changing it (read your password policy).

Prerequisites:

  1. Active aeUser entry exists for user.
  2. User knows user name and current password.

Steps:

  1. Choose "Change" in the menu at the top.
  2. A simple input form is displayed.
  3. Enter your user name, your old password and your new password twice.
  4. Press button to submit the data to the application.
  5. A message is displayed informing you whether the new password was set correctly. In case of an error the input form is displayed again.

Check own password

This function lets a user check his/her password of an Æ-DIR personal user account. Additionally it will also show how long a password is still valid and automatically redirect to Change own password if the password will expire soon.

Hints:

  • Especially you should use this before your on-call duty shift begins to check whether passwords of rarely used accounts for managing production systems are still valid!
  • If your password is soon to expire then change it immediately!

Prerequisites:

  1. Active aeUser entry exists for user.
  2. User knows user name and current password.

Steps:

  1. Choose "Check" in the menu at the top.
  2. A simple input form is displayed.
  3. Enter your user name and your password.
  4. Press button to submit the data to the application.
  5. A message is displayed informing you whether the password was correct and how long it is still valid.

Password reset process

If you forgot your old password or you need initial password for a new account you have to go through this two-step password reset process.

Request password reset

This function lets a user request a password reset in case the user does not know the current password anymore or the current password is expired.

Hints:

  • Starting the password reset procedure does not invalidate your current password.
  • For security reasons the user needs help of a responsible Zone Admin to finish password reset procedure. Hence you should contact a zone admin now (e.g. by phone).
  • This procedure does not work for personal user account serving as mailbox account (hen-and-egg problem)! Ask Zone Admin for password reset procedure in this case.

Prerequisites:

  1. Active aeUser entry has been added for user.
  2. User knows user name and associated e-mail address..

Steps:

  1. Choose "Reset" in the menu at the top.
  2. A simple input form is displayed.
  3. Enter your user name and the accompanying e-mail address.
  4. Press button to submit the data to the application.
  5. A message is displayed informing you that e-mail will be sent to the given address.
  6. you will be sent a new temporary password, check your mailbox now

Finish password reset

This function lets a user request finish password reset process and is typically invoked by clicking on the link in the password reset e-mail requested before.

Hints:

  • The temporary reset password is only valid for a couple of minutes.
  • The 2nd part of the temporary reset password is given to you by the responsible Zone Admin . Contact him now!
  • You must not use old passwords (checked against password history).
  • Your password MUST be sufficiently long (read your password policy).
  • You cannot change your password again for a certain period of time after changing it (read your password policy).

Prerequisites:

  1. 1st part of temporary reset password has been received before via e-mail.
  2. 2nd part of temporary reset password has been told to you by responsible zone admin (e.g. via phone).

Steps:

  1. Click link in the e-mail which already contains 1st part of temporary reset password.
  2. A simple input form is displayed.
  3. Enter your user name (should be already set).
  4. Enter 1st part of temporary reset password (should be already set).
  5. Enter 2nd part of temporary reset password given to you by zone admin.
  6. Enter your new password twice.
  7. Press button to submit the data to the application.
  8. A message is displayed informing you whether the new password was set correctly. In case of an error the input form is displayed again.
