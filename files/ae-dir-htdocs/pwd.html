This document describes how to use the Æ-DIR password self-service web application running on all Æ-DIR provider systems.

Change own password

Prerequisites:

  1. Active aeUser entry has been added for user.
  2. User knows user name and current password.

Check own password

Prerequisites:

  1. Active aeUser entry has been added for user.
  2. User knows user name and current password.

Password reset process

If you forgot your old password or you need initial password for a new account you have to go through this two-step password reset process.

Request password reset

Prerequisites:

  1. Active aeUser entry has been added for user.
  2. User knows user name and associated e-mail address..

you will be sent a new temporary password

Finish password reset

Prerequisites:

  1. 1. part of temporary reset password has been received before via e-mail.
  2. 2. part of temporary reset password has been told to you by responsible zone admin (e.g. via phone).
